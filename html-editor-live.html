プライバシーポリシー - htmlEditorLive

最終更新日：2026年7月1日

本拡張機能について

htmlEditorLive は、HTML ファイルをブラウザ上でビジュアル編集できる社内向けの拡張機能です。外部 API・AI サービスは一切使用せず、すべての編集処理はお使いのブラウザ内で完結します。

収集するデータ

テンプレート機能で要素を保存した場合、その HTML・スタイル・属性のみを chrome.storage.local を用いてお使いのブラウザにローカル保存します。それ以外の個人情報・閲覧履歴・入力内容は収集しません。

外部への送信

編集中の HTML やテンプレートを含め、いかなるデータも外部サーバーへ送信しません。PDF出力・画像の埋め込み処理もすべてローカルで行われます。

権限について

本拡張機能は以下の権限を使用します。

  • storage：テンプレートのローカル保存のため
  • activeTab / scripting / tabs：ユーザーが操作しているタブの HTML を取得し編集対象として読み込むため

取得した HTML は編集目的にのみ使用し、外部への送信や保存は行いません。

第三者への提供

取得・保存したデータを第三者に販売・提供することはありません。
